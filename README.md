# RF Engineer Screening – GDSFACTORY

This repository contains the technical material prepared for an **RF Engineer screening position at GDSFACTORY**.  
It includes design exercises, and layout-related work developed as part of the evaluation process.

---

## Repository Structure

```text
.
├── hello_world/
│   └── Basic introductory project and environment validation
│
├── cells/
│   └── IHP PCell designs for MIM, RFMIM and MOM capacitors and related experiments
│
├── ihp_160g_lna/
│   └── IHP 130nm Low-Noise Amplifier (LNA) design from parametric factories 
│
└── README.md
```

*Note*: 
The LNA design targets the design presented in the repository containing [IHP 130nm April 2025 Tapeout Data](https://github.com/IHP-GmbH/TO_Apr2025/tree/main/160GHz_LNA) - a 160 GHz, 27GHz Bandwidth, 4-stage BJT Technology LNA.
